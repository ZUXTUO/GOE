### 氧气变化规则

- 室外前四层：中和氧气随时间消耗，消耗速率与防护相关（每分钟约等于 21 − 防护）。
- 室外第五层：中和氧气随时间回升。
- 室内燃烧：中和氧气随时间回升。

### 睡眠与恢复

- 睡眠时按设定时长推进时间并消耗饱食与水分。
- 正常环境下，血量随睡眠按床等级提升的对数函数恢复。
- 异常环境（低饱食/低水分/低温/高氧）会触发风险提示或扣血。

### 室内燃烧效果

- 燃烧时室内温度维持约 18℃，氧气维持约 21%。
- 不燃烧时，每约 15 秒室内氧气 +1、温度 −1。
- 室内氧气过高或过低时会按偏差扣减血量。

### 怪物攻击与栅栏规则

- 攻击强度与栅栏等级相关，防御不足时会扣减血量与栅栏耐久。

### 暴击动态调整

- 进入下一层时根据状态对暴击进行调整：高状态增加 5，低状态减少 5。
