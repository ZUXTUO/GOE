### 氧气变化规则

- 室外前四层：中和氧气随时间消耗，消耗速率与防护相关（每分钟约等于 21 − 防护）。
- 室外第五层：中和氧气随时间回升。
- 室内燃烧：中和氧气随时间回升。

### 睡眠与恢复

- 睡眠时按设定时长推进时间并消耗饱食与水分。
- 正常环境下，血量随睡眠按床等级提升的对数函数恢复。
- 异常环境（低饱食/低水分/低温/高氧）会触发风险提示或扣血。

## 探索与随机事件

- 遭遇战概率：随机范围值随天数的对数增加、随楼层加深而降低；范围值内取 0 触发。
- 携带流体：遭遇战范围值 +5，遭遇概率降低。
- 保险箱出现：在允许判定时概率为 1/20。
- 锁耐久：20–80。
- 开锁奖励：物资 5–9。
- 开锁方式：消耗武器并按伤害结算（小刀单次约半伤害、斧头单次全伤害、撬棍单次三倍伤害）。
